# 本地文档检索系统

基于 Rust + egui + tantivy 的本地全文搜索引擎，支持 **PDF / DOCX / TXT** 文档的索引、搜索与实时预览。提供直观的图形界面，支持自动监控文件变动、增量索引、多维度排序以及自定义 PDF 阅读器。

## 功能特性

- 📁 **文件夹索引**：选择本地文件夹，递归扫描支持的文件类型并建立全文索引。
- 🔍 **中文全文搜索**：集成 jieba 分词，支持多关键词搜索，结果按相关度、词频、文件名、修改时间排序。
- 📄 **关键词高亮与上下文预览**：在搜索结果中展示每个关键词命中的位置及前后 **500 字符** 的上下文。
- 🔔 **文件监控**：自动监听文件夹变化，增量更新索引，无需手动重建。
- 🛑 **暂停/恢复/停止**：索引过程可控，适合大量文件场景。
- ⚙️ **自定义 PDF 阅读器**：可配置命令行参数，通过 `{file} {page} {keyword}` 占位符直接跳转到指定位置。
- 🔒 **路径安全**：始终传递文件的绝对路径给外部程序，避免因相对路径导致打开失败。
- 📊 **索引对比**：快速统计新增、修改、删除的文档数量，并一键应用更新。
- 💾 **持久化配置**：自动记忆上次选择的文件夹、索引目录及 PDF 阅读器设置。

## 系统要求

- 操作系统：Windows / macOS / Linux
- 已安装 Rust 工具链（stable 版本即可）
- 若需自定义 PDF 阅读器，请确保对应程序路径正确 C:\Users\<用户名>\AppData\Local\SumatraPDF\SumatraPDF.exe -page {page} {file}
